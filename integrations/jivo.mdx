---
title: "Jivo"
description: "Подключите бота к Jivo чату"
---

<Warning>
    Данная интеграция возможна только в тарифе Jivo Профессиональный.
</Warning>

<Steps>
    <Step title="Запросите подключение бота">
        Напишите [письмо](mailto:info@jivosite.com) в Jivo для подключения бота и получения идентификатора провайдера.

        <Note>Добрый день, хочу подключить бота к своему чату. Адрес сервера `https://api.wikibot.pro/webhook/jivo/<token>`. Предоставьте, пожалуйста, API адрес для отправки запросов.</Note>
    </Step>
    <Step title="Заполните API адрес">
        На [портале](https://app.wikibot.pro) перейдите в раздел **Подключения** своего бота и включите интеграцию Jivo.
    </Step>
    <Step title="Заполните поле API адрес">
       Введите в поле API адрес, полученный от поддержки Jivo.

       <Note>Адрес должен быть в формате `https://bot.jivosite.com/webhooks/<prodiver>/<token>`</Note>
    </Step>
    <Step title="Сохраните интеграцию">
        Нажмите кнопку Сохранить, чтобы включить интеграцию.
    </Step>
    <Step title="Проверьте работу бота">
        Ваш чат-бот готов! Проверьте работу бота в чат-виджете Jivo на вашем сайте.
    </Step>
</Steps>
