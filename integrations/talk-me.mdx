---
title: "Talk-me"
description: "Настройте интеграцию с Talk-me для подключения ответов с использованием ИИ"
---

<Steps>
    <Step title="Получите API-ключ">
        На [портале](https://app.wikibot.pro) перейдите в раздел **Подключения** своего бота и включите интеграцию Talkme.
    </Step>
    <Step title="Сохраните полученный API ключ">
        API ключ необходимо будет использовать при создании сценария в личном кабинете Talkme.
    </Step>
    <Step title="Добавьте бота в личном кабинете Talkme">
        Зайдите в настройки вашего сайта на портале Talkme, перейдите в раздел Боты и создайте нового бота.

        Выберите событие `Стартовал новый диалог` и создайте новый ручной сценарий.

    <Note>
      При необходимости, воспользуйтесь [инструкцией](https://talk-me.ru/kb/nastroiki/boty/ii_boty.html#%D0%98%D0%BD%D1%82%D0%B5%D0%B3%D1%80%D0%B0%D1%86%D0%B8%D1%8F+%D1%81+Wikibot) с портала Talkme.
    </Note>

        <img src="/images/talkme-create-bot.png" alt="talk-me create bot" />
    </Step>
    <Step title="Добавьте бота в сценарий">
        Слева в меню настройки сценария выберите `Сторонний бот` и перетащите на доску сценария.

         <img src="/images/talkme-select-bot.png" alt="talk-me create bot" />
    </Step>
    <Step title="Добавьте интеграцию с Wikibot">
        Нажмите кнопку Добавить интеграцию и выберите Wikibot

        <img src="/images/talkme-select-integration.png" alt="talk-me add wikibot" />
    </Step>
     <Step title="Вставьте сохраненный API ключ">
        В поле API ключ вставьте полученный API ключ из пункта 2.
    </Step>
    <Step title="Выберите базу знаний">
        Добавьте новую базу знаний или выберите уже имеющуюся.
    </Step>
    <Step title="Сохраните интеграцию">
        Сохраните интеграцию, а также сохраните и включите настроенный сценарий.
    </Step>
    <Step title="Проверьте работу бота">
        Ваш чат-бот для обработки обращений готов! Проверьте его работу в любом из каналов Talkme.


    </Step>

  

</Steps>
