<img src="/images/integrations-freshchat.jpg" alt="freshchat integration" />

Вы можете самостоятельно настроить интеграцию с Freshchat и начать обрабатывать обращения пользователей с использованием нашего бота.


<Steps>
  <Step title="Получение Webhook Url">
    На [портале](https://app.wikibot.pro) перейдите в раздел **Интеграции** своего бота и включите интеграцию Freshchat. Скопируйте **Webhook Url**.
  </Step>
  <Step title="Настройка webhook в Freshchat">
    В настройках Freshchat перейдите в раздел Admin settings --> Webhooks for Chat. Включите Webhook. Вставьте полученный на портале Wikibot **Webhook Url** в поле Webhook.
    Нажмите кнопку **Save**. Freshchat автоматически сгенерирует **Authentication** - публичный ключ. Скопируйте его.
    <img src="/images/integrations-freshchat-enable-webhook.jpg" alt="freshchat integration" />    
  </Step>
  <Step title="Настройка публичного ключа">
    Полученный публичный ключ вставьте в настройках интеграции Wikibot в поле **Публичный ключ**.
  </Step>
  <Step title="Настройка API запросов">
    В настройках Freshchat перейдите в раздел Admin settings --> API Settings. Нажмите кнопку **Generate** для генерации нового API ключа. Скопируйте его в поле **API ключ** в
    настройках интеграции. Скопируйте **Your chat URL** в поле **Чат URL**. 
    Больше информации о настройке API Settings Вы найдете в [документации Freshchat](https://developers.freshchat.com/api/#authentication).
    <img src="/images/integrations-freshchat-enable-api.jpg" alt="freshchat integration" />    
  </Step>
  <Step title="Задайте агента для перевода">
    Выберите агента, на которого бот будет переводить вопросы без ответов либо
    запросы Оператора.
  </Step>
  <Step title="Выберите группы">
    Выберите группы Freshchat, в рамках которых будет отвечать бот. Если ни одна группа не выбрана, то бот будет отвечать на все сообщения.
  </Step>
  <Step title="Проверьте работу бота">
    Ваш чат-бот готов! Проверьте работу чат-бота в любом из каналов Freshchat.
  </Step>
</Steps>

Если у вас остались вопросы по интеграции, [напишите нам](https://t.me/use_wikibot).
