## Мастер подключения

<Steps>
    <Step title="Создайте бота">
        Для создания бота перейдите по адресу https://app.tomleto.pro/bots/new.

        Задайте имя для вашего бота, выберите интеграцию **Verbox** и нажмите кнопку **Создать**.

        <img src="/images/integrations-verbox.jpg" alt="verbox integration" />
    </Step>
    <Step title="Создайте API-ключ">
        На следующем шаге вам будет автоматически сгенерирован API-ключ.

        Для копирования ключа нажмите кнопку **Копировать**. Нажмите кнопку **Далее** для переходу к следующему шагу.

        <img src="/images/integrations-verbox-api-key.jpg" alt="verbox integration api key" />
    </Step>
    <Step title="Завершите настройку">
        Вы попадёте на экран завершения настройки, после чего будете перенаправлены на страницу бота.

        <img src="/images/integrations-success.jpg" alt="integration success" />
     </Step>
     <Step title="Проверьте API-ключ">
        На странице бота выберите пункт меню **API ключи** и перейдите на страницу с API-ключами. Здесь вы можете просмотреть список и добавить новый ключ, либо скопировать один из существующих.

        <img src="/images/integrations-verbox-api-keys.jpg" alt="verbox integration api keys" />
     </Step>
     <Step title="Активируйте бота в Verbox">
        Для завершения интеграции перейдите на сайт [Verbox](https://verbox.ru/) и завершите настройку бота.
     </Step>
</Steps>

## Ручной режим

<Info>
    Если вы создали бота через телеграм, то вы можете самостоятельно выпустить API-ключ в веб интерфейсе и настроить бота в Verbox.
</Info>

<Steps>
    <Step title="Войдите в личный кабинет">
        Зайдите в [личный кабинет](https://app.tomleto.pro) Wikibot через телеграм-аккаунт, на который вы регистрировали бота. Созданный бот  автоматически привяжется к вашему аккаунту.
    </Step>
    <Step title="Создайте API-ключ">
        Перейдите в раздел **API ключи**. Нажмите кнопку **Добавить API ключ**. В открывшемся диалоговом окне введите ключ, либо нажмите кнопку **Сгенерировать** и затем нажмите кнопку **Сохранить**.

        <img src="/images/integrations-verbox-add-api-key.jpg" alt="verbox integration add api key" />
    </Step>
    <Step title="Скопируйте API-ключ">
        Наведите мышку на строчку таблицы с добавленным API-ключом и нажмите на иконку копирования.

        <img src="/images/integrations-verbox-copy-api-key.jpg" alt="verbox integration copy api key" />
    </Step>
     <Step title="Активируйте бота в Verbox">
        Для завершения интеграции перейдите на сайт [Verbox](https://verbox.ru/) и завершите настройку бота.
     </Step>
</Steps>
