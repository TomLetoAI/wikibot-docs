---
title: "Verbox"
description: "Настройте интеграцию с Verbox для подключения ответов с использованием ИИ"
---

<Steps>
    <Step title="Получите API ключ">
       На [портале](https://app.wikibot.pro) перейдите в раздел **Подключения** своего бота, включите интеграцию Verbox и скопируйте API ключ.
    </Step>
     <Step title="Активируйте бота в Verbox">
        Для завершения интеграции перейдите на сайт [Verbox](https://verbox.ru/) и завершите настройку бота, используя скопированный API ключ.
     </Step>
</Steps>
