## Начало работы

Базовый URL: https://api.wikibot.pro/api.

Взаимодействие с API осуществляется при помощи **POST** и **GET** запросов. Убедитесь что POST запрос содержит заголовок Content-Type: `application/json`.

## Асинхронный режим

Поддерживаются синхронный и асинхронный механизмы ответа. Все запросы являются синхронными по умолчанию. Для перехода на асинхронный механизм необходимо задать **webhook URL**.

<Info>
  Синхронные запросы возвращают ответ, либо ошибку в контексте текущего
  HTTP-запроса.
</Info>

<Info>
  Асинхронные запросы сразу же возвращают пустой HTTP 200 ответ и вызывают
  webhook URL с результатами выполнения запроса.
</Info>

## Авторизация

Все запросы должны содержать заголовок **Authorization**.

```
Authorization: my_api_key
```

Создать API ключ можно в [личном кабинете](/lk/api-keys).
