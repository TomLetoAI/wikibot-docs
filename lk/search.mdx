title: "Поиск"

---

<img src="/images/lk-search.jpg" alt="search" />

На странице **Поиск** можно использовать бота в качестве семантического поиска по вашей Базе Знаний. Найденные страницы и файлы будут проранжированы по релевантности
вашему поисковому запросу.

Для интеграции поиска во внешние системы можно воспользоваться соотвествующим методом API. Подробное описание смотрите [здесь](/api-reference/endpoint/get-search)

<Warning>Стоимость одного поиска - 0.1 кредита.</Warning>
