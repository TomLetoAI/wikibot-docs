На этой странице вы можете настроить основные параметры бота:
- Имя бота и язык;
- Расписание;
- Шаблоны сообщений;
- Фразы;
- API ключи;
- Опасная зона.

## Имя бота и язык
Введите имя бота в поле **Имя**, выберите язык, на котором будет отвечать бот, и нажмите **Сохранить**.

<img src="/images/lk-settings-name.jpg" alt="settings bot name" />

## Расписание
Определите рабочие часы и выходные дни, в соответствии с которыми будет работать ваш бот. В выходные дни бот будет использовать отдельные шаблоны сообщений.

<Info>
    Используйте для составления режима работы ваше **местное время**. Мы автоматически сделаем необходимые преобразования.
</Info>

Чтобы задать единое расписание на все дни, укажите интервал в поле **По умолчанию**.

Чтобы задать особенное время для какого-то **конкретного** дня, переопределите время работы для этого дня.

Чтобы отметить определённый день как выходной, нажмите галочку **Выходной**.

<img src="/images/lk-settings-work-hours.jpg" alt="settings work hours" />

## Шаблоны сообщений

<Warning>
    Шаблоны сообщений содержат важные системные поля, такие как `{message}` и `doc("...", "...")`.
    
    Убедитесь, что вы не удаляете их либо делаете это сознательно, иначе бот не будет выдавать часть ответа, связанную с определённым полем. Также убедитесь, что вы сможете восстановить эти поля, если понадобится их вернуть.
</Warning>

<Info>
    `{message}` — служебное поле для непосредственного ответа бота.

    `docs("сообщение до списка", "сообщение после списка")` — функция для добавления списка статей, которые использовались для генерации ответа. 
    Сообщения могут быть пустыми. Если документов не найдено, то функция не добавит никаких сообщений.
</Info>

Задавайте шаблоны сообщений для различных сценариев.

Для применения изменений отредактируйте нужный шаблон и нажмите кнопку **Сохранить**. Обновленный шаблон сообщения применится в демо-чате справа.

<img src="/images/lk-settings-templates.jpg" alt="settings templates" />

## Фразы

Расширьте предустановленные списки фраз, по которым бот определяет сообщения, которые нужно **пропустить**, а также **приветствия** и **благодарности**. Введите в текстовые поля слева соответствующие фразы, резделяя их переносом строки, и нажмите кнопку **Сохранить**.

<img src="/images/lk-bot-settings-phrases.jpg" alt="settings phrases" />

Вы можете проверить, как будет реагировать бот на определенное сообщение. Для этого введите пример пользовательского сообщения в текстовое поле справа. Результат обработки сообщения ботом будет выведен ниже.

## API ключи

### Просмотр списка

В данном разделе вы можете просмотреть список созданных API ключей.

Для просмотра или копирования API ключа наведите указатель мыши на нужную строчку и нажмите соответствующую иконку.

<img src="/images/lk-api-keys.jpg" alt="api keys" />

### Создание ключа

Для создания нового API ключа нажмите кнопку **Добавить API ключ**. В открывшемся диалоговом окне введите ключ, либо нажмите кнопку **Сгенерировать** и затем нажмите кнопку **Сохранить**.

<img src="/images/integrations-verbox-add-api-key.jpg" alt="verbox integration add api key" />

### Удаление ключа

Нажмите кнопку удаления API ключа, которая находится справа в строке с ключом.

## Опасная зона

<Warning>
  Выполнение описанных действий приведёт к удалению данных. Эти действия
  невозможно отменить.
</Warning>

Для полного удаления бота и связанных данных нажмите **Удалить этого бота** и подтвердите операцию, введя имя бота в диалоговом окне, и нажав **Продолжить**.

<img src="/images/lk-danger-zone-confirm.jpg" alt="danger zone confirm dialog" />

После удаления данных вы увидите оповещение с кнопкой перехода на [список ботов](/lk/bot-list).
