Аналитика показывает сводную информацию по неделям.

- **Неделя** — дата начала недели.
- **Всего тикетов** — общее количество обработанных тикетов. Обратите внимание, это не количество вопросов, а количество уникальных диалогов.
- **Закрыто ботом** — общее количество тикетов, за исключением тикетов, на которые бот не смог дать ответ и перевёл на оператора. Тикеты с переводом на оператора с помощью первой линии считаются закрытыми ботом.
- **Закрыто ботом, %** — процентное отношение закрытых ботом тикетов. Цвет числа показывает изменение по сравнению с прошлой неделей. Зелёный — показатель вырос, красный — показатель упал.

Также дополнительно можно скачать **.xlsx**-отчёты о работе бота.

- **Общий отчёт** — общий отчёт по месяцам: количество отправленных запросов и оставшиеся кредиты.

- **Общий отчёт по дням** — отчёт по дням: общее количество обращений и ответов по базе.

- **Детальный отчёт** — подробный отчёт по запросам, включая тип запроса, дату и время, сам вопрос и ответ.
