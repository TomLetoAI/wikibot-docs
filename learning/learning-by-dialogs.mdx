---
title: "Обучение по диалогам"
description: "Обучите бота с помощью накопленной переписки с пользователями"
---

Обучение по диалогам позволяет автоматически сгенерировать базу знаний в виде набора статей и использовать полученную БЗ для работы бота.
Генерация производится на основе выгрузки диалогов из вашего хелпдеска. Напишите [нам](https://t.me/use_wikibot) для получения услуги.
